# Star_Wars_Survey
Cleaning Star Wars Survey Data

## Installation  

You will need `python 3` installed to run __Star Wars Survey.ipynb__.


```
% git clone git@github.com:kthotav/Star_Wars_Survey.git

% cd Star_Wars_Survey

% jupyter notebook

```
